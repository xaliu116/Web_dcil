# 手写体文本识别
1. 手写体文本识别采用的库为torch和gradio，torch的版本为1.9.0

2. 环境配置：
conda env create -f pytorch.yml

3. 手写体文本检测识别模型参数文件：
[百度网盘](https://pan.baidu.com/s/1uOw4pywJA_c3UaLj_Ja4dw?pwd=dcil 
) 提取码：dcil
，粘贴到项目output目录下
